This function performs the following steps: 1. Creates input files for batch inference, including formatting text based on metadata, generating message prompts,

and preparing JSONL files.

  1. Uploads the generated input files to an AWS S3 bucket based on the configuration.

  3. Downloads and processes the output files generated during the batch job.

  4. Returns the processed batch results for further utilization.
