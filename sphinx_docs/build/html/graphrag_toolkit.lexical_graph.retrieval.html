add_entity(entity: ScoredEntity)[source]¶

Adds an entity to the list of entities, either merging it with an existing entity or appending it as a new entry.

If the entity already exists in the list (based on a matching entityId), its score is incremented by the score of the added entity. Otherwise, the new entity is appended to the list.

Args:

entity (ScoredEntity): The scored entity to add or merge with an existing entity.
